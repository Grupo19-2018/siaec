Crear nuevo usuario

Para crear un nuevo usuario siga los siguientes pasos:

  1. De clic en Nuevo usuario, en el cual se desplegara el formulario para crear al nuevo usuario.
  2. Seleccione el tipo de rol que será el usuario.
    Dependiento del tipo de rol se activaran las casillas.
  3. Llene todos los campos necesarios, en especial aquellos que son obligatorios.
  4. De clic en guardar.

Regresar
Editar usuario

Para editar un usuario siga los siguientes pasos:

  1. Encuentre en la tabla de usuarios, el usuario que quiera editar.
  2. Busque en la columna de opciones, la opcion editar, si no le aparencen opciones puede que su usuario no este autorizado.
  3. Una vez de clic en la opcion editar le aparecera el formulario para poder editar al usuario.
  4. Modifique los campos pertinentes.
  5. De clic en guardar.
Regresar
Eliminar usuario

Para eliminar un usuario siga los siguientes pasos:

  1. Encuentre en la tabla de usuarios, el usuario que quiera eliminar.
  2. Busque en la columna de opciones, la opcion eliminar, si no le aparencen opciones puede que su usuario no este autorizado.
  3. Una vez de clic en la opcion eliminar de clic.
  4. Le aparecera un cuadro desplegable en el cual se le pedira confirmar la accion.
  5. Una vez confirmada la accion el cuadro se cerrar y el usuario se eliminara.
